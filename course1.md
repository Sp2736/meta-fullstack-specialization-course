## Course Overview

The course introduced the fundamentals of:

* How the Internet and Web work
* Front-End, Back-End, and Full-Stack roles
* HTML, CSS, JavaScript — structure, styling & behavior
* Web accessibility, DOM, developer tools
* Responsive design and UI frameworks (Bootstrap)

The course was divided into **4 modules**, each ending with assessments, labs, programming tasks and graded assignments — all successfully completed.

* Developer tools and debugging
* IDE setup, Visual Studio Code & extensions
* Core Internet Concepts + Knowledge Checks

Completed Tasks:

* Command-Line Web Protocol Lab
* Website Inspection + Editing Exercise
* Multiple graded practice assignments (Score: 100%)
